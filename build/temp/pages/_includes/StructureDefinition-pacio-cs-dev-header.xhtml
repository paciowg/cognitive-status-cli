A record of an assistive device such as a hearing aid being used by a patient during a cognitive status assessment. This extension based on the DeviceUseStatement resource provides a way for a BundledCognitiveStatus resource to show records of device usage during an assessment.

This profile builds on DeviceUseStatement.
